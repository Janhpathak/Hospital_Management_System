
# Hospital Management System (Java Servlet)

Java GUI-based Hospital Management System using Servlets, JSP, and JDBC.

## Features
- Add Patient
- View Patient
- MVC Architecture
- Servlet-based GUI

## Tech Stack
Java, Servlet, JSP, JDBC, MySQL, Tomcat

## How to Run
1. Import project in IntelliJ as Maven/Java Web Project
2. Configure Tomcat Server
3. Create MySQL database `hospital`
4. Run on server
